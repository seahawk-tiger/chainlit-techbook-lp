Chainlit技術書同人誌の特設ページです。

## ディレクトリ構成

```
├── index.html          # メインページ
├── css/style.css       # スタイルシート
├── js/main.js          # JavaScript
├── assets/images/      # 画像（書影・OGP等）
└── reference/          # 参考資料（.gitignore対象）
```

## ローカルプレビュー
